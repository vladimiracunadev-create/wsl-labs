# Cheatsheet - Comandos Linux

```bash
pwd                 # ver ruta actual
ls                  # listar archivos
ls -la              # listar con detalles
cd carpeta          # entrar a carpeta
mkdir carpeta       # crear carpeta
touch archivo.txt   # crear archivo vacío
cat archivo.txt     # mostrar archivo
nano archivo.txt    # editar archivo
cp origen destino   # copiar
mv origen destino   # mover/renombrar
rm archivo          # eliminar archivo
rm -r carpeta       # eliminar carpeta
chmod +x script.sh  # dar permiso de ejecución
```
