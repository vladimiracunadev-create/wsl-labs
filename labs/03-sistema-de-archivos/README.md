# 03 - Sistema de archivos en WSL

## Objetivo

Entender la diferencia entre el sistema de archivos Linux y el acceso al disco de Windows.

## Ruta Linux recomendada

```bash
mkdir -p ~/proyectos
cd ~/proyectos
```

## Acceso a disco C de Windows

```bash
cd /mnt/c
ls
```

## Abrir carpeta Linux en Explorer

```bash
explorer.exe .
```

## Recomendación

Para proyectos Linux, Node, Python, PHP o bases de datos, trabajar preferentemente en:

```bash
~/proyectos
```

No en:

```bash
/mnt/c/Users/...
```
