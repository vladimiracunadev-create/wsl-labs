# 00 - ¿Qué es WSL?

WSL significa **Windows Subsystem for Linux**. Permite ejecutar distribuciones Linux dentro de Windows sin levantar una máquina virtual tradicional manualmente.

## Idea principal

WSL sirve para trabajar con herramientas Linux desde Windows:

- Bash.
- Git.
- Python.
- Node.js.
- PHP.
- PostgreSQL.
- Nginx.
- Apache.
- SSH.
- Automatización con scripts.

## Qué no es WSL

WSL no es Docker.
WSL no es una máquina virtual tradicional de escritorio.
WSL no es un reemplazo directo de producción para un servidor Linux real.

## Qué se puede aprender en este repo

- Crear y administrar distros Linux.
- Montar servicios dentro de WSL.
- Automatizar configuraciones.
- Exportar e importar entornos.
- Comparar WSL con Docker y máquinas virtuales.
